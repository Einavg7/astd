The Fourier transform

The Fourier transform decomposes any signal into the weighted sum of a number of sin and cos functions with different frequency:

\[ s_N(t) = a_0/2 + \sum_{n=1}^{N} \big( a_n \cos(\frac{2\pi n t}{P}) + b_n sin(\frac{2\pi n t}{P}) \big) \]

with \(P\) the period (frequency \(1/P\)), and \(N\) the length of the series. The Fourier transform, or spectrum, or periodogram, depicts amplitude against frequency.

artificial data:

pure sinus data, unsmoothed (raw) spectrum:

n = 1000
x = pi * 1:n
s1 = sin(x / 100)
plot(s1, type = 'l')


plot(spectrum(s1))

smoothed spectrum:

plot(spectrum(s1, spans = c(3,5)))

higher frequency sinus:

s2 = sin(x / 10)
plot(s2, type = 'l')


plot(spectrum(s2))

smoothed curve:

plot(spectrum(s2, spans = c(3,5)))

superposition of both:

plot(s1+s2, type = 'l')


plot(spectrum(s1+s2, spans = c(3,5)))

adding noise to them

ns = s1 + s2 + rnorm(n)
plot(ns, type = 'b')
plot(spectrum(ns))

smoothed spectrum:

plot(spectrum(s1+s2+rnorm(n), spans = c(3,5)))
